# FocusCycle Watch App

A comprehensive Apple Watch app for Yoga, Pranayama, and Meditation with streak tracking and customizable alerts.

## Features

### 🧘‍♀️ Yoga Timer
- Heart rate monitoring during sessions
- HealthKit integration for mindful minutes
- Streak tracking
- Customizable alert settings

### 🌬️ Pranayama (Breathing Exercises)
- **Anulom Vilom**: 4-4-4-4 breathing pattern
- **Ujjayi**: 6-2-6-2 breathing pattern  
- **Bhastrika**: 1-1-1-1 rapid breathing pattern
- Customizable timing for each phase (inhale, hold, exhale, hold)
- Phase-specific alert settings
- Streak tracking per pattern

### 🧠 Meditation Timer
- 5, 10, 15, 20, 30 minute sessions
- Background timer support
- Streak tracking
- Customizable alert settings

## Technical Features

- **Streak Management**: Tracks daily streaks for all activities
- **Alert System**: Haptic and sound alerts with customizable settings
- **HealthKit Integration**: Saves mindful minutes to Health app
- **WatchOS Optimized**: Designed for Apple Watch with proper sizing
- **Accessibility**: Supports Dynamic Type and accessibility features

## Requirements

- watchOS 9.0+
- Xcode 14.0+
- Apple Watch

## Installation

1. Clone the repository
2. Open `FocusCycle.xcodeproj` in Xcode
3. Select your Apple Watch target
4. Build and run on your Apple Watch

## Project Structure

- `FocusCycle Watch App/` - Main app source code
- `FocusCycle Watch AppTests/` - Unit tests
- `FocusCycle Watch AppUITests/` - UI tests
